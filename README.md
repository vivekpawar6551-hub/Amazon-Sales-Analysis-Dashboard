# Amazon Sales Analysis Dashboard

Interactive Power BI dashboard for analyzing Amazon product sales, reviews, and category performance.

## Features
- Total Sales KPI
- YTD Sales Analysis
- QTD Sales Analysis
- Product Sold Tracking
- Customer Reviews Analysis
- Monthly Sales Trend
- Weekly Sales Performance
- Product Category Analysis
- Top 5 Products by Sales
- Top 5 Products by Reviews

## Tools Used
- Power BI
- DAX
- Power Query

## Insights
- Men Shoes generated the highest YTD sales
- Sales performance increased significantly in the last quarter
- Product reviews crossed 19M+
- Weekly sales peaked around week 50
